# PCL-TDGCN
This repo provides a demo for the "Prototypical Contrastive Learning with Temporal Dynamic Graph Convolutional Network for EEG-based Emotion Recognition" on the SEED and HIED datasets.


### Requirements
* `Python 3.9`
* `Pytorch 1.1.0`
